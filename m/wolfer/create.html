Create Your Own Wolfer Game

Follow these steps to create your own Wolfer game. You can submit it to be listed on the official Wolfers page.

1. Fill out the template to describe your game

2. Copy this LLM prompt

3. Paste the prompt into a new LLM session

Here are some free LLM chatbots you can use:

  • ChatGPT
  • Microsoft Copilot
  • Google Gemini

4. Copy the code your LLM generated

Be careful to copy the code by itself without any other part of the chatbot conversation.

5. Paste the code here

6. Load your code and test your game

7. Submit your code to be listed on the official Wolfers page

Once you're happy with your game, Copy Game Code and submit it here.
